# Allegro-Pong
/*Allegro Pong is a 2D 2P Ping Pong game using the Allegro Library version 4.4 including two paddles with a ball and a BITMAP background that resembles a realistic Ping Pong table. Both players are controlled by 4 different keyboard inputs: 1P - W, S. 2P - Up and Down. The objective of the game is to see who gets to 5 points first and bonus rickroll plays in the background.*/
